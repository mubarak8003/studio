# **App Name**: RecoupPro

## Core Features:

- Trade Record Entry: Log individual win/loss amounts with date and timestamp.
- Recovery Ratio Configuration: Allow users to specify exactly how many future winning sessions should distribute the recovery of the current total loss.
- Smart Stake Calculator: Automatically calculates the next stake amount needed based on the user's defined recovery period and historical loss data.
- AI Strategy Coach Tool: An AI-powered tool that analyzes your loss patterns and recovery settings to suggest if your current recovery plan is sustainable based on recent hit rates.
- Session Management: Start and stop sessions to track net performance over a specific period of activity.
- Historical Persistence: Save all session data and recovery settings to a cloud database for access across devices.

## Style Guidelines:

- A dark, high-contrast theme focused on financial precision. Primary color is a focused Electric Blue (#647cf0) representing logic. Background is a deep Obsidian (#16181d).
- Accent color is a crisp Sky Blue (#2699e1), used for progress bars and calculated result highlights.
- Font pairing: 'Space Grotesk' for high-impact numbers and headlines to provide a tech-forward, algorithmic feel, paired with 'Inter' for clear, objective UI labels and body text.
- Clean, card-based dashboard with large numeric displays to prioritize immediate legibility of the next stake amount.
- Minimalist geometric icons focusing on directional trends (upward/downward) and balance.
- Subtle counter-up animations on stake amount changes and soft-glow transitions when switching from loss to recovery mode.
